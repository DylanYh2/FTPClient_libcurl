# FTPClient_libcurl
包含功能为：   
1、向Ftp服务器上传文件   
2、从Ftp服务器下载特定文件  
3、从Ftp某个文件夹里面下载所有文件   
4、远程在FTP服务器指定位置创建文件夹   
5、查看FTP服务器指定目录内所有文件名   
解决了FTP上传/下载文件时，文件名有中文，有特殊字符时无法上传/下载的问题。

libcurl编译方式：见csdn，链接如下：
http://t.csdnimg.cn/jo3AY
